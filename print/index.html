Yes there is

CSS Custom Properties

Also known as

'CSS variables' or 'cascading variables'

Let's see how these can be used in our sample markup.

Inherit

They do inherit. This means that if no value is set for a custom property on a given element, the value of its parent is used.

Override

Can be overriden

.two {
  color: var(--my-var, red); /* Red if --my-var is not defined */
}

Fallback

Can give fallback values using `var()` function when the given variable is not defined

.three {
  background-color: var(
    --my-var,
    var(--my-background, pink)
  ); /* pink if --my-var and --my-background
  are not defined */
}

✅ The right way

.three {
  background-color: var(
    --my-var,
    --my-background,
    pink
  ); /* Invalid: "--my-background, pink" */
}

❌ The wrong way

Can have multiple fallback values as well

:root {
  --indent-size: 10px;
​
  --indent-xl: calc(
    2 * var(--indent-size)
  );
  --indent-l: calc(
    var(--indent-size) + 2px
  );
  --indent-s: calc(
    var(--indent-size) - 2px
  );
  --indent-xs: calc(
    var(--indent-size) / 2
  );
}
